INTRODUCTION
------------

User Avatar intends to improve the default User blank Avatar to a dynamic new 
Avatar fetched for a Avatar providing site like http://avatars.adorable.io/.

REQUIREMENTS
------------

No special requirements.

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.

CONFIGURATION
-------------

 * After Installation go to url DOMAIN/admin/config/user_avatar/adminsettings or
   Admin -> Configuration -> Media -> User Avatar Configuration
 * Add required configuration there. Default prefilled configuration are 
   taken from http://avatars.adorable.io/. For more configuration details check 
   http://avatars.adorable.io/#demo
